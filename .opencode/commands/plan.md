# 代码库分析与计划

只调用 `testcase-planner` Skill，扫描项目代码库并生成代码生成计划。

## 使用方式

```
/plan <粘贴结构化测试步骤>
```

## 适用场景

- 已有标准格式的测试步骤（如之前 refine 的输出）
- 只想看生成计划，不立即生成代码
- 需要评审代码复用方案

## 输出

详细的代码生成计划，包含：
- 已有可复用资源清单
- 需要新建的文件和方法
- 执行顺序

这份输出可以作为 `testcase-coder` 的输入。
